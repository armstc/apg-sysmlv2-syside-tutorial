# APG SysML v2 Tutorial with Syside

The "APG-files / Tutorials folder" contains the following files:

1. `01_README.md` - This file that you are reading.
2. `02_Your_notes.md` - An editable file for you to write notes during your training session.
3. `03_Flashlight_StarterModel.sysml` - An example simple SysML v2 model for you to review.

And it contains the following folders:

1. `Backups` - This contains several files that represent what your simple model should look like after completing different sets of exercises.
2. `Snippets` - This contains several files that you will use in the exercises to avoid typing everything from scratch yourself.

## Basic Functionality Cheat-sheet

- **To visualize the current model**:
  - In the top right corner click `SysIDE Logo → Visualize current file`
  - Alternatively, use the keyboard shortcut `Ctrl+Shift+V` on a selected
    `.sysml` file
  - You can go from the diagram element to the source code by right-clicking
    on an element and choosing "Go To Source Definition"

- **Syntax & Semantic Validation**: As the model is being edited, it will be
  checked for any errors in real time. All areas of issue will be highlighted
  with red squiggly lines underneath. To get a description of the issue, simply
  hover over the highlighted elements.

- **Go to declaration**: Navigate to element definitions by holding `Ctrl` and
  clicking on the desired element

- **Code snippet auto-completion**: Use `Tab` for code snippet auto-completion

- **To open document outline:**
  - In the bottom left corner, click `> Outline`
  - Note: The outline is also visible by default at the top of the editor
    screen, right next to the name of the open file.

- **To format the current model**:
  - Right click and select `Format Document`
  - Alternatively, use the keyboard shortcut `Ctrl+Shift+I`
